Instructions

Scroll down and complete each step in order to move from setup to 21 cm signal analysis.

Scroll to begin ↓

STEP 1

Installation

Install required software and drivers, including RTL-SDR dependencies and project scripts.

Confirm your SDR is recognized before starting any recording sessions.

STEP 2

Data Collection

Prepare dish alignment, verify amplifier and filter wiring, and run baseline recordings.

Record with consistent gain settings and save metadata for each observation.

STEP 3

Processing

Run the processing pipeline to convert raw SDR captures into analyzable spectra.

Compare output against reference frequency ranges around the 21 cm line.

STEP 4

Validation

Repeat observations where possible and compare trends over multiple sessions.

Log anomalies so your team can refine hardware configuration and calibration.
